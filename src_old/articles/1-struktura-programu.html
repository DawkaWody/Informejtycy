Struktura programu w C++

Zaczynając przygodę z programowanien, warto zrozumieć podstawową strukturę programu w C++, ponieważ wszystkie programy będą bazowały na podobnym szablonie. W tym artykule przedstawimy krok po kroku strukturę prostego programu w C++.

Podstawowa struktura programu

#include <iostream>
using namespace std;

int main()
{
    // Twój kod tutaj

    return 0;
}

Każdy program w C++ składa się z kilku podstawowych elementów:

Polecenie #include na początku programu

Na początku programu często znajduje się specjalne polecenie zaczynające się od #include. Takie polecenia informują komputer, które dodatkowe funkcje i narzędzia będą używane w programie. Najczęściej spotykanym poleceniem tego typu jest:

#include <iostream>

<iostream> to biblioteka, która pozwala korzystać z funkcji do wyświetlania tekstu na ekranie (np. za pomocą cout) oraz do wczytywania danych od użytkownika (np. przy użyciu cin).

Dodanie przestrzeni nazw

Aby uprościć korzystanie z elementów standardowej biblioteki C++, można dodać polecenie:

using namespace std;

Dzięki temu nie trzeba będzie poprzedzać nazw takich jak cout czy cin prefiksem std::.

Funkcja main

Funkcja main jest punktem wejścia każdego programu w C++. To właśnie od niej zaczyna się wykonanie programu.

int main()

int oznacza, że funkcja zwraca wartość całkowitą (zwykle 0, jeśli program zakończył się poprawnie). Nawiasy () mogą zawierać argumenty, ale dla prostych programów są puste.

Blok kodu {}

Kod, który ma być wykonany przez funkcję main, znajduje się w nawiasach klamrowych {}. To tzw. ciało funkcji.

{
    // Twój kod tutaj
}

Instrukcja return

Na końcu funkcji main zwykle znajduje się instrukcja return, która zwraca wartość całkowitą. Wartość ta informuje system operacyjny o zakończeniu programu.

return 0;

0 oznacza, że program zakończył się sukcesem. Jeśli program napotka błąd, można zwrócić inną wartość (np. 1).

Jak uruchomić program w C++?

Aby skompilować program, np. pomocą g++, w wierszu poleceń należy wpisać:

g++ -o program nazwa_pliku.cpp
gdzie -o to flaga po której podaje się nazwę pliku wykonywalnego, program to nazwa pliku wykonywalnego, a nazwa_pliku.cpp to Twój plik z kodem. Aby go uruchomić, należy wpisać: 
./program
na Linuxie lub: 
program.exe
na Windowsie.
